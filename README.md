# STM32 Bare Metal Driver Development

This repository is dedicated to learning and developing bare-metal drivers on the STM32 platform while building a deeper understanding of ARM Cortex-M microcontroller architecture.

The primary goal is to move beyond vendor-provided abstraction layers and gain practical experience working directly with hardware through memory-mapped registers, peripheral documentation, startup code, and linker scripts.

## Purpose

This repository serves as a structured workspace for experimenting with, implementing, and documenting low-level embedded systems concepts as they are learned.

Areas of focus include:

- Register-level programming
- GPIO driver development
- Interrupts and exception handling
- UART communication
- Timers and PWM
- ADC and DMA
- SPI and I2C interfaces
- ARM Cortex-M architecture
- Startup code
- Linker scripts
- Embedded driver design and abstraction

## Hardware

- STM32F446RE Nucleo Board
- ARM Cortex-M4

## Development Environment

- STM32CubeIDE
- GCC ARM Toolchain
- Git

## Notes

This is an active learning repository. Code, notes, experiments, and driver implementations will be added as new concepts are explored and understood.
